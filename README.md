📌 Project Overview

This project focuses on analyzing customer churn in a telecommunications company using Power BI. The dashboard provides actionable insights into customer behavior, churn trends, demographics, tenure patterns, contract types, payment methods, and internet services to help businesses improve customer retention.

🎯 Objectives
Analyze customer churn rate and retention patterns.
Identify key factors influencing customer churn.
Visualize customer demographics and tenure distribution.
Provide data-driven insights for business decision-making.
Build an interactive Power BI dashboard for stakeholder reporting.

📈 Key Insights:
Month-to-month contract customers show higher churn rates.
Customers using Fiber Optic internet services tend to churn more frequently.
Electronic Check users exhibit higher churn compared to other payment methods.
Customers with shorter tenure are more likely to leave the service.

🚀 Business Recommendations:
Introduce loyalty programs for new customers.
Encourage long-term contracts through discounts and incentives.
Improve customer support for high-risk customer segments.
Offer personalized retention campaigns for churn-prone customers.
